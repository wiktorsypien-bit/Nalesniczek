# Nalesniczek — Hermes AI System

Personalny agent AI zbudowany przez Wiktora Sypienia.
Hermes działa autonomicznie na MacBooku, zarządza zadaniami, integracjami i codziennym życiem.

## Co to jest Hermes

Hermes to osobisty agent AI który:
- Zna twoje cele, rutyny i kontekst życiowy
- Działa autonomicznie w zdefiniowanych granicach
- Integruje się z Gmail, Spotify, Apple Notes, Google Calendar
- Pilnuje priorytetów i przypomina o tym co ważne
- Chroni prywatność rodziny i dane wrażliwe
